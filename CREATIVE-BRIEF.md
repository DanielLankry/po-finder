### Phase 2: Refresh (Week 3-4)

Based on Phase 1 data, create 2-3 new variations:

- New visual: same concept, different map view
- New copy variation: "X אנשים חיפשו עסקים קרובים היום — העסק שלך לא שם"

- New visual: different price comparison (less than a bus ticket, etc.)

## Design Guidelines for Canva

### Colors
- Primary: #059669 (buttons, accents, checkmarks)
- Secondary: #047857 (hover/dark variant)
- Background: #FAFAF7 (warm white)
- Text: #1C1917 (stone-900)
- Subtle text: #78716C (stone-500)

### Typography
- Headlines: Secular One (or Heebo Bold as Canva alternative)
- Body: Heebo Regular
- Direction: RTL always

### Logo Usage
- Always include "פה" logo on every creative
- Use /public/logo.png as the source file
- Position: bottom-right (RTL = visual anchor on reading-end side)
- Size: minimum 60x60px

### Image Rules
- Keep text < 20% of image area (Meta penalizes text-heavy images)
- Use real screenshots of the Po platform when possible
- Avoid stock photos that look too "corporate" — keep it local and authentic
- Always include the green (#059669) somewhere — brand recognition
- Reference /public/og-image.png for the brand's social sharing aesthetic
